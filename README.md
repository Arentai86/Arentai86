# OpenZen

Practical AI products and privacy-first digital tools for real-world tasks.

I’m Artem, the founder and builder behind OpenZen — an independent product studio from Engelskirchen, Germany.

OpenZen builds small, focused applications that turn complex everyday problems into clear next steps: finding help in Germany, searching for medical practices, sorting waste correctly, and creating AI-assisted engineering workflows.

## What OpenZen builds

### HelpZen
A privacy-first help navigator for people in Germany.

HelpZen helps users find relevant social, government, emergency, and community support without getting lost in bureaucracy.

### MedZen
Doctor-search assistance for Germany.

MedZen helps people find relevant medical practices faster. It is not a diagnosis tool and does not replace professional medical advice.

### LifeZen
AI-assisted waste-sorting guidance for Germany.

LifeZen makes Mülltrennung simpler: take a photo of an item and get a clear recommendation for the right bin category. It is free on the App Store, has no advertising, no tracking, and is designed for practical everyday use.

App Store: https://apps.apple.com/de/app/lifezen/id6760695188
Support: https://github.com/Arentai86/lifezen-support

### OpenZen Studio
AI-assisted CAD and product design workflows for macOS.

OpenZen Studio explores how AI can help turn ideas into structured models, prototypes, and engineering tasks while keeping the human in control.

### OpenZen Remote
A privacy-first remote control layer for AI agents.

OpenZen Remote connects mobile apps, local tools, and agent workflows through a safer gateway with authentication, audit logs, capability approvals, and human review.

### OpenClaw for macOS
AI-generated native macOS launcher for the OpenClaw local AI workspace.

OpenClaw for macOS turns a local AI workspace into a Mac menu bar app with guided setup, localhost runtime control, provider authentication, bundled skills, diagnostics, and a downloadable DMG.

Repository: https://github.com/Arentai86/openclaw-mac
Download: https://github.com/Arentai86/openclaw-mac/releases/latest

### Hermes Agent for macOS
AI-generated native macOS launcher for the local Hermes Agent dashboard.

Hermes Agent for macOS starts and monitors a local Hermes dashboard, bundles official skills, bootstraps the runtime, stores credentials in Keychain, and ships as a downloadable Mac DMG.

Repository: https://github.com/Arentai86/hermes-agent-mac
Download: https://github.com/Arentai86/hermes-agent-mac/releases/latest

## AI-native build statement

OpenZen is openly experimenting with AI-generated software.

The OpenClaw for macOS and Hermes Agent for macOS launcher repositories were generated by AI agent workflows under human direction, review, and product judgment. The point is transparency: publish the work, let people inspect it, improve it, fork it, and challenge it.

## Principles

- Practical before flashy
- Privacy by default where possible
- Clear interfaces for complex tasks
- Human control over AI actions
- Small tools that can grow into real systems
- Useful automation with visible logs and permissions

## Current focus

- Building OpenZen Remote: mobile control for local AI agents and workflows
- Improving HelpZen and MedZen for real-world use in Germany
- Developing OpenZen Studio as an AI-assisted engineering workspace
- Strengthening security: SSRF protection, audit trails, capability cards, and secret redaction
- Turning OpenZen into a clear, trustworthy product ecosystem
